bai 1:
git commit --amend: để sửa commit message qua giao diện VIM
git commit --amend -m"<message>": đổi trực  tiếp commit message ko đi qua giao diện VIM
git restore --staged <file> : chuyển file từ vùng staging về vùng working directory
git reset HEAD~1: chuyển 1 commit ở reposity về vùng working directory
